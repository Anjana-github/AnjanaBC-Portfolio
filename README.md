# Anjana B C — Portfolio

Personal portfolio website for **Anjana B C**, Android Developer with ~4 years of experience building high-traffic OTT streaming applications for millions of users across India and 15 South American countries.

🔗 **Live site:** https://github.com/Anjana-github/AnjanaBC-Portfolio.git


## About

This is a single-page portfolio built with plain HTML, CSS, and JavaScript — no frameworks, no build step. It covers:

- Summary & core competencies
- Technical & soft skills
- Professional experience
- Key projects (industry + academic)
- Education & certifications
- Contact details with an embedded, downloadable CV

## Tech

- HTML5 / CSS3 / vanilla JS
- Google Fonts: Space Grotesk, Inter, IBM Plex Mono
- No dependencies, no build process — just static files

## Running locally

Clone the repo and open `index.html` directly in a browser:

```bash
git clone [https:https://github.com/Anjana-github/AnjanaBC-Portfolio.git
cd AnjanaBC-Portfolio
open index.html   # or double-click the file
```

## Deployment

Hosted via **GitHub Pages**:

1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main`, folder: `/ (root)`
